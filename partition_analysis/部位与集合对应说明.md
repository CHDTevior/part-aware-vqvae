## 四、计算方法说明

### 1. 关节激活序列

对每个关节 j，取其对应的 12 维特征（ric 3 维 + rot 6 维 + vel 3 维），计算**每帧的 L2 范数**作为该帧的“激活强度”：

```
activation[t] = ||feat[t]||_2  = sqrt(sum(feat[t]^2))
```

将所有 motion 的激活序列拼接，得到关节 j 的全局激活序列 `(N_frames,)`。

### 2. 关节间相关性矩阵

构建矩阵 `(N_frames, 22)`，每列是某关节的激活序列。计算**皮尔逊相关系数**：

```
corr[i,j] = Pearson(activation_i, activation_j)
```

得到 22×22 的相关性矩阵。相关性高表示两关节在运动时**同步变化**（一起动、一起静）。

### 3. 层次聚类

- **距离定义**：`dist[i,j] = 1 - |corr[i,j]|`
  - 相关性越高 → 距离越近 → 更可能被分到同一簇
- **聚类方法**：average linkage（平均链接）
- **切割**：将树状图切割为 K=6 个簇

### 4. 物理含义

- **Part 0（身体核心）**：根、脊柱、肩、肘等关节在大部分动作中协同变化，相关性高，聚为一簇。
- **Part 1（双膝）**：左右膝在行走、蹲起等动作中往往同步，单独成簇。
- **Part 2/3（左右小腿）**：踝、趾与膝分开，可能与步态中脚部独立运动有关。
- **Part 4/5（左右臂末端）**：腕、手与肩肘分离，手部精细动作与躯干运动模式不同。
